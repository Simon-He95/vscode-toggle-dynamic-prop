通过快捷键去将当前 prop,切换动态和非动态模式,因为可能这个属性有多种类型,不一定是字符串也可能是对象等等,可以随意变更, 比如原本使用的网络地址, 突然需要根据环境使用变量拼接的形式, 或者字符串突然不适合表达, 需要拼接变量等等的转换, 这个时候就可以使用这个插件来快速切换
